### Setup Python

- Instalasi `uv`

```
# On macOS and Linux.
curl -LsSf https://astral.sh/uv/install.sh | sh
```

- Instalasi `uv`
```
uv python install 3.13
```

- Check instalasi python
```
python3.13 --version
```
