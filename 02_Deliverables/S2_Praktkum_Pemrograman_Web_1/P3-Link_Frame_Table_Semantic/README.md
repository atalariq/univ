# Pertemuan 3 - Link, Ifram, Table, dan Semantic

## Tugas Praktikum

1. Buat folder baru di laptop/pc dengan nama pertemuan_2
2. Buka text editor yang ada di laptop/komputer
3. Buat file baru dengan nama index.html, kemudian simpan di dalam folder pertemuan_2
4. Buat tabel dengan header:

  - Nama
  - Usia
  Isi dengan data diri Anda.

5. Buat struktur semantic HTML menggunakan tag berikut:

  - `<header>`
  - `<nav>`
  - `<main>`
  - `<section>`
  - `<article>` (minimal 2)
  - `<aside>`
  - `<footer>`
  Tampilan tidak harus sama persis dengan gambar, yang penting struktur HTML benar.

## Langkah Praktikum

1. Link biasa, link gambar, link ke window baru, dan link internal
2. Frameset (horisontal dan vertikal)
3. Frameset dengan atribut no-resize = “true”
4. Frame inline
5. Membuat table
6. Sel kosong dalam tabel
7. Sel tabel yang diperlebar
8. Menambahkan latar belakang ke table
9. Membuat layout website sederhana menggunakan table
10. Membuat HTML Semantic Hader, Nav
11. Membuat HTML Semantic Article, Figure, Figcaption
12. Membuat HTML Semantic Main, Section
13. Membuat HTML Semantic aside
