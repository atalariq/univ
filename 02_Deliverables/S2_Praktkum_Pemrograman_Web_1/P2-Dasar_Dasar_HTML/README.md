
# Pertemuan 2 - Dasar-Dasar HTML

## Tugas Praktikum

1. Buat folder baru di laptop/PC dengan nama pertemuan_1.
2. Buka text editor (VSCode/Notepad/Sublime/dll).
3. Buat 8 file HTML sesuai daftar latihan. Setiap file harus memiliki struktur dasar HTML yang benar.
4. Simpan setiap file dengan nama yang sesuai dan ekstensi ".html" (contoh: latihan1_paragraf.html) di dalam folder pertemuan_1.
5. Buka file tersebut di browser dan pastikan tampil dengan benar.

## Daftar latihan

1. Paragraf dan line break
2. Background color dan heading
3. Garis horizontal dan komentar tersembunyi
4. Pemformatan teks
5. Teks yang di-preformat
6. Arah teks dan quotation
7. Jenis-jenis list
8. List bersarang dan list definisi
